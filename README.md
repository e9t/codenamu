# 그랬던 그가
[코드나무](http://twitter.com/codenamu) 공공데이터 캠프 프로젝트!

## 취지
정치인이나 유명인이 했던 과거 행동이나 발언을 지금과 비교 하는 전시용 페이지나 인포그래픽. 좋은 것은 칭찬하고 나쁜 것은 잊지 말자는 목적입니다. ^^
